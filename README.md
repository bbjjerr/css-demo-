### 先定义子元素，子元素写好高度和宽度可以的定义父盒子

| 写法                  | .box1 宽度规则 |
| --------------------- | -------------- |
| display: block        | 父元素 100%    |
| display: flex         | 父元素 100%    |
| display: inline-block | 子元素宽度     |
| display: inline-flex  | 子元素宽度     |
| width: fit-content    | 子元素宽度     |
